Часы с эмоциями (yamaflip)

## Описание проекта
Проект представляет собой простые часы с графическим интерфейсом, которые отображают текущее время, дату, день недели и случайные эмоции, меняющиеся каждые 5 минут.

## Функциональность
- Отображение текущего времени (часы и минуты) с мигающим разделителем
- Отображение текущей даты (день, месяц)
- Отображение дня недели
- Случайная эмоция, которая меняется каждые 5 минут
- Графический интерфейс на базе Tkinter

## Используемые технологии
- Python 3.x
- Стандартные библиотеки:
  - `tkinter` - для графического интерфейса
  - `datetime` - для работы с датой и временем
  - `random` - для генерации случайных эмоций

## Установка и запуск

### Требования
- Установленный Python 3.x
- Доступ к стандартным библиотекам Python

### Запуск программы
1. Скопируйте код из файла `yamaflip.py`
2. Сохраните его в файл с расширением `.py`
3. Запустите файл

## Структура кода

### Основные функции:
1. `clock()` - генерирует строку с текущим временем, датой и случайной эмоцией
2. `clicked()` - заглушка для обработки кликов (в текущей реализации не используется)
3. `update_text()` - обновляет текст в графическом интерфейсе каждую секунду

### Особенности реализации:
- Время обновляется каждую секунду
- Эмоция меняется каждые 5 минут (или при достижении 60 минут)
- Разделитель между часами и минутами мигает (меняется на пробел каждую секунду)
- Форматирование вывода с выравниванием по центру

## Интерфейс
- Окно размером 210x120 пикселей
- Заголовок окна: "yamaflip"
- Текст отображается в виде:
  ```
  ЧЧ:ММ          Эмоция
  
  Месяц:        Название месяца
  День:         Число месяца
  День недели:  Название дня недели
  ```
